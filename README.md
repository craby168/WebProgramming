# Todo JS App

A simple vanilla JavaScript todo list application.

## Features

- Add new todos with title and description
- View todos in a scrollable list
- Expand/collapse todo descriptions by clicking the title
- Delete todos with the red delete button
- Checkboxes for selection (no functionality implemented yet)
- Fixed header with input fields

## Usage

1. Open `index.html` in a web browser.
2. Enter a new todo title and description.
3. Click "Add Todo" or press Enter to add it.
4. Click on a todo title to show/hide its description.
5. Click the delete button to remove a todo.

## Development

- Managed with Yarn
- No build process required, pure frontend

## Troubleshooting

- Ensure a modern web browser is used.
- If styles don't load, check file paths.
